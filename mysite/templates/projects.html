Week 7: PWA Transformation & Offline Mode

üì¶ GitHub Repo
This week transformed the standard Flask website into a fully functional Progressive Web App (PWA) that works offline and can be installed on devices:

  • Web Manifest: Created manifest.json to enable "Add to Home Screen" functionality with custom icons and theme colors.
  • Service Worker: Implemented sw.js to cache the "App Shell" (HTML, CSS, JS) and handle network requests.
  • Offline Capability: The app now loads instantly even without internet by serving cached assets via the Fetch API.
  • Flask PWA Routing: Updated flask_app.py to serve the Service Worker from the root scope (`/sw.js`) to control the entire application.
  • Extra Feature (Offline Alert): Added a real-time JavaScript notification system that detects network loss and warns the user when they are offline.

Key Features:
Service Worker Web Manifest Offline Caching Installable Network Detection
